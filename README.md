# Karate Judge & Diplomas Panel

Одностраничное веб-приложение для ведения соревнований по каратэ: судья управляет категориями и боями, дипломщик отмечает выдачу дипломов.

## Запуск

Откройте `index.html` в современном браузере — сборка или сервер не требуются.

## Возможности

- Создание соревнований и категорий (круговой турнир или олимпийская сетка на 4 участника).
- Автогенерация боёв, пошаговое заполнение результатов и расчёт итоговых мест.
- Дэшборд дипломщика для отметки, что дипломы по завершённым категориям выписаны.
